# Chicken Vegetable Dumplings Recipe Website

A simple multi-section recipe website built using **HTML only**.  
This project explains how to make **Chicken Vegetable Dumplings** from scratch, including:

- Dumpling wrapper dough
- Dumpling filling
- Cooking instructions
- Homemade chilli oil dipping sauce

## Project Goal

This project was created to practice:

- HTML document structure
- Semantic HTML tags
- Navigation links
- Images
- Lists (ordered and unordered)
- Internal page linking using IDs

## Features

✔ Semantic HTML layout  
✔ Navigation menu with anchor links  
✔ Recipe sections with ingredients and instructions  
✔ Images for visual presentation  
✔ Footer section

## How to Run

1. Open the project folder in VS Code.
2. Open `index.html`.
3. Right click → **Open with Live Server**
   (or double click the file to open in browser).


## Website Sections

### 1. Header
- Website title
- Short introduction
- Featured dumpling image

### 2. Main Components
Navigation menu for:
- Dumpling Wrapper
- Dumpling Filling
- Cooking Method
- Chilli Oil Sauce

### 3. Dumpling Wrapper Dough
Instructions and ingredients for preparing wrapper dough.

### 4. Dumpling Filling
Steps for preparing the chicken and vegetable filling.

### 5. Cooking Dumplings
Guide for steaming and pan frying.

### 6. Chilli Oil Sauce
Simple dipping sauce recipe.
